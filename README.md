# 🌐📊 OpenDataMarket

**OpenDataMarket** is a **full-stack web application** developed as a **specialized graduation project (team of 3)**.  
The system provides an **online dataset marketplace** that allows users to buy, sell, and manage datasets securely, integrating **escrow-based payment**, **real-time communication**, and **AI-powered features**.

---

## 📌📖 Project Overview

- 🎓 **Project type**: Specialized course / Graduation project  
- 👥 **Team size**: 3 members  
- 🏗️ **Architecture**: Client–Server (RESTful API)  
- 🎯 **Objective**: Build a secure, scalable, and user-friendly platform for dataset trading and management  

---

## 🚀✨ Core Features

- 👤 User registration and authentication with role-based access (Buyer / Seller / Admin)
- 📦 Dataset marketplace: upload, browse, search, and purchase datasets
- 💰 Secure transaction flow using **internal wallet and escrow mechanism**
- 📑 Order management and transaction tracking
- 💬 Real-time chat between buyers and sellers
- 🤖 AI-powered dataset search using natural language queries
- 🧠 AI chatbot for user support and system guidance
- 🛡️ AI-based file analysis to detect malware or scam content
- 🧑‍💼 Admin dashboard for managing users, datasets, and transactions
